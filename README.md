# 🚀 Project Overview

Urban Happiness Index Analysis is a **data science project** that explores how different urban lifestyle factors influence the happiness level of people living in cities.

The system performs:

✔ Statistical Analysis  
✔ Correlation Study  
✔ Data Visualization  
✔ Interactive Dashboard using **R Shiny**

This project demonstrates how **data-driven insights can help understand urban well-being and quality of life.**
